# WhatsApp Chat Analyzer with Spam Detection

## 📌 Project Overview

WhatsApp Chat Analyzer is a Machine Learning and NLP based project developed using Python. The application analyzes WhatsApp chat data and generates useful insights such as user activity, timelines, emoji usage, word clouds, and spam message detection.

The project uses a Logistic Regression Machine Learning model to classify messages as Spam or Normal.

---

## 🚀 Features

* Total Messages Analysis
* Total Words Analysis
* Monthly Timeline
* Daily Timeline
* Most Active Users
* Word Cloud Generation
* Emoji Analysis
* Media Shared Analysis
* Link Shared Analysis
* Spam Message Detection
* Interactive Dashboard using Streamlit
